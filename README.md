# AI SmartHire

אב טיפוס אינטרקטיבי לאפליקציית Web מבוססת AI להכנה מותאמת אישית לראיונות עבודה.

## מה יש באתר
- דף פתיחה מקצועי ומעוצב
- הסבר על תהליך העבודה
- טופס להדבקת קורות חיים ותיאור משרה
- העלאת קובץ TXT/MD/CSV
- ציון התאמה דינמי
- חוזקות, המלצות לשיפור, שאלות ראיון ומכתב מקדים
- עיצוב רספונסיבי למחשב ולמובייל

## איך להעלות ל-GitHub Pages
1. פתחי Repository חדש ב-GitHub.
2. העלי את הקבצים: `index.html`, `style.css`, `script.js`, `README.md`.
3. היכנסי ל-Settings → Pages.
4. בחרי Branch: `main` ו-Folder: `/root`.
5. שמרי ותקבלי קישור לאתר.

## הערה
הניתוח באתר מדמה מנוע AI בצד הלקוח. בפרויקט אמיתי ניתן לחבר API של Gemini דרך שרת Backend.
